# RoadCare 24 - Emergency Towing & Roadside Assistance Website

A modern, mobile-first one-page website for RoadCare 24, a towing and emergency roadside assistance company operating across Sri Lanka.

## Features

- **Modern Design**: Clean, professional design with red (#dc2626), white, and dark gray color palette
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Sticky Header**: Always visible navigation with logo and phone number
- **Floating Call Button**: Mobile-friendly floating call button for quick access
- **Smooth Animations**: Section reveal animations and smooth scrolling
- **Accessibility**: ARIA labels, semantic HTML, and proper focus states
- **Form Validation**: Client-side form validation for the partner form

## Sections

1. **Hero Section**: Prominent phone number and large "Call Now" button
2. **Services Section**: Four service cards (Emergency Towing, Jump Start, Tyre Change, Fuel Delivery)
3. **Why Choose Us**: Trust-building features (24/7 availability, licensed drivers, etc.)
4. **Testimonials**: Customer reviews and testimonials
5. **Partner With Us Form**: Contact form for potential partners
6. **Footer**: Company information and contact details

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
